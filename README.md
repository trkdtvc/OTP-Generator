# OTP Generator

A React-based OTP (One-Time Password) generator that creates a 6-digit code with a 5-second countdown timer.

## Features
- Generates a random 6-digit OTP
- 5-second countdown with live updates
- Button disabled while timer is active
- Expiration message when the OTP expires

## Tech Stack
- React (Hooks)
- JSX
- HTML & CSS

## Purpose
Built as a practice project to demonstrate state management, timers, and conditional rendering in React.
